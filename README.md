##  Описание

Консольная система для управления заметками в ИТ-инфраструктуре VPN-сервиса. Позволяет администраторам создавать, хранить и управлять заметками на любом устройстве ИТ-ландшафта, мониторить загрузку системных ресурсов и вести журналы безопасности.

##  Технологии

- **Python 3.14**
- **PostgreSQL 18** (через psycopg2)
- **pytest 9.1.0** (автоматическое тестирование)
- **Git** (контроль версий и автообновления)
- **PyCharm 2026** (IDE)

##  Установка

### 1. Установка PostgreSQL
Скачайте с https://www.postgresql.org/download/windows/

### 2. Создание базы данных
- Вручную
- С помощью SQL-запроса
### 3. Создание тестовых пользователей
python creat_users.py
### 4. Настройка подключения
```bash
DB_CONFIG = {
    'host': 'localhost',
    'database': 'vpn_notes',
    'user': 'postgres',
    'password': '5555555555',
    'port': '5432'
}
```
## Запуск программы
python main.py

## Запуск тестов
pytest Tests/ -v

## Команды

### Авторизация

| Команда       | Описание      | 
| ------------- | ------------- |
| login         |Вход в систему |
| logout        |Выход из сессии| 

### Заметки
| Команда       | Описание      | 
| ------------- | ------------- |
| addNewNote "текст"| Создать заметку  |
|viewNotes  | Просмотр заметок  | 
|deleteNote|Удалить заметку|

### Мониторинг (только admin)
| Команда       | Описание      | 
| ------------- | ------------- |
| start_watchdog|Запустить фоновый мониторинг |
| stop_watchdog |Остановить мониторинг| 
|viewMetrics|Просмотр истории метрик|

### Администрирование (только admin)
| Команда       | Описание      | 
| ------------- | ------------- |
| viewLogs      |Вход в систему |
|viewUsers      |Выход из сессии| 
|deleteUser     |Удалить пользователя|

### Общие команды
| Команда       | Описание      | 
| ------------- | ------------- |
| help          |Справка по командам |
| exit          |Завершение работы| 

### Роли
| Команда       | Описание      | 
| ------------- | ------------- |
| user         |Создание, просмотр, редактирование, удаление своих заметок |
| admin       |	Полный доступ + управление пользователями + просмотр логов + мониторинг|
